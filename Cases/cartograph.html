We all use Figma or Sketch for interface design, but have you ever wondered what tool you need to create maps?

If you are a big company, such as Yandex, you should have great tools and invest in your processes. That's why we developed Cartograph in 2018. It is our internal tool for creating and publishing maps. Here is a sneak peek of my work and Cartograph interface.

Layer management and group editing

Comprehensive color management: create, store, and select map styles

History and map settings

Zoom logic for map customization

JSON Editor

Efficient branch management and release system like GitHub
